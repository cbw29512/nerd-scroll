NERD SCROLL FILES

This folder is for your own Nerd Scroll content.

Use these folders:

1. Homemade Scrollers
   Put text files, stories, notes, books, Bible passages, ASCII scenes, or packs you make yourself here.

2. Purchased Scrollers
   Put Nerd Scroll packs you bought or downloaded here.

Open the Nerd Scroll app from the desktop shortcut, then use Add Pack... to load a file.

Nerd Scroll reads text only. It does not run the text as code.
