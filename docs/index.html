Portable Windows terminal effect tool

Nerd Scroll Typing Tool

Drop in a text file, run the app, and watch your command window type it out like a human. Great for coding visuals, streams, demos, videos, fake terminal scenes, and desktop ambience.

How to use it

1. Download

Click the download button and grab the latest zip file from GitHub Releases.

2. Extract

Right-click the zip file, choose Extract All, and open the extracted folder.

3. Add text

Paste your scene text into DROP_TEXT_FILE_HERE.txt.

4. Run

Double-click the app and it will type your text into the command window.

5. Record

Use it as a background, stream scene, tutorial shot, or video effect.

6. Stop

Close the command window when you are done.

Tips for better scenes

  • Use short lines so the terminal looks clean on video.
  • Add blank lines between sections for pacing.
  • Mix real code with comments to make it readable.
  • Use timestamps for system-log scenes.
  • Use headings like STARTUP, SCAN, BUILD, DEPLOY, or MISSION LOG.
  • Keep one scene style per text file for the best visual effect.

Important Windows note

Windows may show a warning because this is an unsigned independent app. Only run software from sources you trust. This tool is portable, does not install services, does not require internet, and only reads the text file placed beside it.
